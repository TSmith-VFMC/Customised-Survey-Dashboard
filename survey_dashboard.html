Data & Technical Information
▼
⚙️ How It Works
This dashboard is a single-page HTML app that runs entirely in your browser — no server required.
Survey questions are embedded from survey_questions.json, generated by the Python script.
The Python script (survey_generator.py) analyses source text using NLP to extract key themes, findings, and statistics.
Three question types are produced: Likert scale (attitudes), Multiple choice (categories), and Open-ended (qualitative).
📥 Data Sourcing
Paste or load any report or PDF text into survey_generator.py to regenerate questions tailored to new content.
The script uses frequency analysis and signal-word detection to surface the most important themes.
Questions are written to survey_questions.json — edit this file manually to refine wording.
💾 Response Storage
Responses are saved to browser localStorage during your session — they persist if you close and reopen the tab.
Use Export JSON for system integration or archiving.
Use Export CSV to open responses directly in Excel.
💡 Tips & Hints
✅Run survey_generator.py with a new text file any time you want fresh questions from a different report.
✅Manually edit survey_questions.json to fix or improve question wording before sharing.
✅The progress bar in the header shows how many questions you've answered.
✅Your answers auto-save — if you accidentally close the browser, reopen and they'll still be there.
✅CSV exports open directly in Excel — use columns D for answers.
⚠️Clearing browser data will erase saved responses — export first.
